## Build Setup

```bash
# ติดตั้ง package
$ yarn install

# ลองรัน dev บนเครื่องเรานะครับจะรันอยู่บน localhost:3000
$ yarn dev

# build ในแบบ production
$ yarn build
$ yarn start

# ทำโปรเจคแบบ static ไม่เอา ssr
$ yarn generate
```

โปรเจค Nuxt + Express + Sequlize เพื่อการเรียนรู้
